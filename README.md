# 🏦 Streamlit Bank App

A simple Bank Management System built using Python and Streamlit that allows users to manage bank accounts with an interactive UI.
---

## 🚀 Features
- Create a new account
- Deposit money
- Withdraw money
- View account details
- Update account information
- Delete account

---

## 🛠 Tech Stack
- Python
- Streamlit
- JSON (for data storage)
- Object-Oriented Programming (OOP)
